## Responsive Behavior

### Desktop (default)
- 3-column grid for preset buttons
- Full-width custom time input with button on the right
- Full-width reset button

### Mobile (smaller screens)
- Preset buttons may stack to 2 columns or 1 column
- Custom time input and button may stack vertically
- All text remains readable

## Interactive States

### Preset Buttons
- Default: Blue background
- Hover: Darker blue
- Click: Brief visual feedback, then status message appears

### Custom Time Input
- Default: Gray border
- Focus: Blue ring appears
- Invalid: No visual change (validation on submit)

### Reset Button
- Default: Amber (override) or Gray (no override)
- Hover: Slightly darker shade
- Click: Status message appears, button color changes

## Time Display Updates

The "Current Time" display updates every second:
- Shows real time ticking if no override
- Shows static overridden time if override is active
- Format: "Weekday, Month Day, Year at Hour:Minute AM/PM"
- Example: "Sunday, January 7, 2024 at 10:30 PM"

## Accessibility

- All buttons have clear labels
- Color is not the only indicator (text labels included)
- Keyboard navigation supported
- Focus states visible
- Semantic HTML structure

## Browser Compatibility

- HTML5 datetime-local input
- LocalStorage API
- Modern JavaScript (ES6+)
- Works in all modern browsers (Chrome, Firefox, Safari, Edge)
